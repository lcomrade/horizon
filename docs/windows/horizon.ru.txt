Horizon Manual
Horizon 0.6
24.09.2021


НАЗВАНИЕ
    Horizon - минималистичный WEB-сервер для передачи данных по протоколу HTTP


ОПИСАНИЕ
    Horizon - это простая программа, выполняющая единственную функцию - передачу данных по протоколу HTTP.
    Несмотря на свою простоту, она поддерживает TLS шифрование и пользовательский шаблон для HTTP-страницы.


СИНТАКСИС
    horizon [ПАРАМЕТРЫ]...


ПАРАМЕТРЫ
    Чтобы узнать больше о доступных аргументах командной строки, выполните: 'horizon -help'


ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ
    LANG
      Указывает язык программы. Например ru_RU - русский, en_US - английский.


ФАЙЛЫ
    Horizon хранит свои настройки в конфигурационных файлах в формате JSON.
    Horizon использует файлы конфигурации, расположенные в директориях %APPDATA%\horizon\ и C:\ProgramData\horizon\.

    Более подробно параметры конфигурационного файла описаны в файле configure.md. URL:
    https://github.com/lcomrade/horizon/blob/main/docs/configure.md


СИМВОЛИЧЕСКИЕ ССЫЛКИ
    Horizon поддерживает символические ссылки.
